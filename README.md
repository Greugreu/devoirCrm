# Devoir CRM

Symfony 6 avec PHP 8.0

## Stack

**Front-end** : Twig

**Back-End** : Symfony 6.0

## Instalation

Installer d'abord symfony sur la machine depuis le site de [symfony](https://symfony.com/download)

```bash
  $ composer install
  $ npm run dev
  $ symfony server:start
```
