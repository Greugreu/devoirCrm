# Devoir CRM

Symfony 6 avec PHP 8.0

## Stack

**Front-end** : Twig

**Back-End** : Symfony 6.0

## Instalation

[Symfony](https://symfony.com/download)

```bash
  $ composer install
  $ npm run dev
  $ symfony server:start
  
  http://127.0.0.1:8000/home
```
